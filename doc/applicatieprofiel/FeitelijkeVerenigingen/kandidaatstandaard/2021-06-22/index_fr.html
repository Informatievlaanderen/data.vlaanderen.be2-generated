Ce document décrit un profil d’application, dans ce cas Feitelijke Verenigingen - Applicatieprofiel. Ce profil d’application répond à la question de savoir comment le modèle de domaine correspondant peut être appliqué dans la pratique. De plus, les limitations (cardinalité, listes de codes) sont expliquées et les termes correspondants (RDF) sont listés.

Résumé

Ce profil d’application vous permet de décrire ActualAssociations.

Une association d’établissement des faits est une organisation qui, comme les autres associations, a un but désintéressé. La différence est qu’une association de fait n’est pas enregistrée en tant qu’entité juridique, elle n’est qu’une association parce que les membres entreprennent des activités ensemble.

Les associations réelles peuvent être reconnues, ce qui leur donne un caractère formel sans être une entité juridique.

Afin de pouvoir utiliser les services publics, par exemple le prêt d’équipements sportifs par une administration locale, il peut suffire de fournir quelques informations de base, mais parfois cette reconnaissance est également nécessaire.

Le traitement de la demande de reconnaissance et de la reconnaissance finale est effectué par un organisme public. Afin d’obtenir la Reconnaissance, certaines preuves doivent être fournies par l’Association Actuelle sur la base de critères spécifiques.

Le profil de candidature permet également de décrire qui est membre de la Fact-Association, et/ou qui est responsable (et quelle est cette responsabilité).

Enfin, il est possible de décrire quels services publics ont été sollicités par la Fact-Association.

Le profil d’application réutilise les termes et la manière dont ils y sont appliqués à partir du profil d’application OSLO Organization et du catalogue de services OSLO.

Une description plus formelle des termes peut être trouvée dans le vocabulaire OSLO-FactualSocieties.
