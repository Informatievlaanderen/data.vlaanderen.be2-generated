Ce document décrit un profil d’application, dans ce cas Sol (profil d’application). Ce profil d’application répond à la question de savoir comment le modèle de domaine correspondant peut être appliqué dans la pratique. De plus, les limitations (cardinalité, listes de codes) sont expliquées et les termes correspondants (RDF) sont listés.

Résumé

Le profil d’application du sol explique comment la terminologie de ce domaine doit être utilisée pour l’échange de données sur les observations de sol.

Le profil d’application s’appuie sur le profil d’application OSLO Sensors and Sampling, entre autres.

Ce sous-domaine concerne l’échantillonnage et l’observation des sols.

L’échantillonnage est effectué à des endroits du sol (parfois plus spécialisés comme forage de sol ou puits profilé).

En plus de l’analyse d’échantillons de sol, le profil du sol est déterminé et le sol dans son ensemble est classé selon différents systèmes de classification des sols.

Le profil d’un sol à la suite d’une observation se compose d’horizons et de couches qui peuvent également être échantillonnés ou observés séparément.

Dans ce domaine également, il y a des observations qui peuvent être décrites de manière générique.
