Ce document décrit un profil d’application, dans ce cas Réseau de surveillance des eaux souterraines (profil d’application). Ce profil d’application répond à la question de savoir comment le modèle de domaine correspondant peut être appliqué dans la pratique. De plus, les limitations (cardinalité, listes de codes) sont expliquées et les termes correspondants (RDF) sont listés.

Résumé

Le profil d’application du réseau de surveillance des eaux souterraines explique comment la terminologie de ce domaine doit être utilisée pour échanger des données sur les observations des eaux souterraines.

Le profil d’application s’appuie sur le profil d’application OSLO Sensors and Sampling, entre autres.

Ce sous-domaine concerne la surveillance du niveau et de la qualité des eaux souterraines.

Le niveau est mesuré dans ce que l’on appelle des puits de surveillance, généralement avec des piézomètres ouverts ou fermés placés dans ces puits.

La surveillance peut être effectuée manuellement ou automatiquement à l’aide de capteurs.

La qualité est déterminée par l’observation de divers paramètres de qualité sur des échantillons d’eau souterraine obtenus dans les puits et les piézomètres.

Les piézomètres sont organisés en réseaux de surveillance.
