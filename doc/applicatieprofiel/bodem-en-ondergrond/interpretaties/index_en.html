This document describes an application profile, in this case Soil and subsoil interpretations (application profile). The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

Summary

The OSLO Soil & Subsurface Interpretations application profile explains how the terminology from the corresponding vocabulary should be used to exchange data on specialized Observations of the type of Interpretation regarding Soil & Subsoil.

The application profile builds on the OSLO Sensors and Sampling application profile, among other things.

Interpretations are Observations that involve a personal or reasoned judgment. This judgement is based on theoretical knowledge and/or processing of other Observations.

In the context of Soil and Subsoil, it is often a question of Interpretations of stratification: stratigraphic or lithological characteristics of the layers or anthropogenic or geotechnical layers that can be distinguished. This application profile focuses on such Interpretations, but other types of Observations can also be classified as Interpretations.

How layering varies with depth is described by means of Spatial Ranges, i.e. a geometry (depth from-to) and a value with the characteristics of the layer per layer.

These characteristics depend on the type of layer. For example, it is about the type of soil, the geological formation, the water lining, etc. They are usually described by means of values from a code list.

As mentioned above, the input for the Interpretations can be other Observations, e.g. generic Observations described with the application profile Sensors and Sampling, or more specific ones such as those from the application profile OSLO Soil and Subsurface Observations or from the application profile OSLO CPTs.

The Observation Procedure and Input classes can be used to refer to these other Observations.
