This document describes an application profile, in this case Soil and subsurface observations (application profile). The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

Summary

The Soil & Subsurface Observations application profile explains how the terminology from the corresponding vocabulary should be used to exchange data on specialised Soil & Subsurface Observations.

The application profile builds on the OSLO Sensors and Sampling application profile, among other things.

For Observations that are generic, e.g. those of the so-called recognition (Soil type, Humus content, Volumetric mass, etc.), the classes of Sensors and Sampling are sufficient.

This is not the case for the Observations in this application profile that usually take place in a lab with specialized equipment: determining the Grain Distribution or geotechnical characteristics such as Permeability, Compressibility or Shear Resistance.

For these analyses, the necessary classes and characteristics are provided, usually specializations of the class Measurement series because the result varies with a certain parameter, e.g. the proportion of a certain grain diameter in a soil sample or the deformation of a sample with increasing pressure.

The analyses usually relate to soil samples, but for the description of these we refer to the application profile Soil drilling.

This application profile does not concern Interpretations, i.e. Observations that involve a reasoned personal judgment, which are dealt with in the application profile OSLO Soil and Subsurface Interpretations.
