This document describes an application profile, in this case Cultural Participation - Application Profile. The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

Summary

This application profile makes it possible to describe cultural participation, the Participation in an Activity and can be used, for example, for the program 'Goal-Oriented Digital Transformation'.

An Activity is an Action that is deliberately performed by an Agent with the intention of producing a cultural, social or material effect. In this application profile, we refer to this Agent as a Realizer who can be a Person or Organization.

An Activity can be participated in by a Participant. This relationship is called a Participation. A Transaction and Ticket can then be linked to this. Feedback may also be given about the Participation.

For the Participant in an Activity, a generic or personal Participant Profile can be exchanged in which matters relating to Cultural Participation, General and Sociodemographic Characteristics are contained.

The Realisator offers the Activity and can carry out Promotion through a Channel. This class can be further specified to Organizer.

About the Activity there is information about which Thing is being carried out and by which Contractor, but also about the Price Info and Booking Info.

The application profile reuses terms and the way they are applied from the application profile OSLO-Organization, OSLO-CulturalHeritage:Event and OSLO-CulturalHeritage:Object, OSLO Person, OSLO-Consent, OSLO Grant and OSLO-ActualAssociation.

A more formal description of the terms can be found in the OSLO-Cultural Participation vocabulary.
