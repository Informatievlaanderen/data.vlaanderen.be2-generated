Ce document décrit un profil d’application, dans ce cas Participation culturelle - Profil de demande. Ce profil d’application répond à la question de savoir comment le modèle de domaine correspondant peut être appliqué dans la pratique. De plus, les limitations (cardinalité, listes de codes) sont expliquées et les termes correspondants (RDF) sont listés.

Résumé

Ce profil d’application permet de décrire la participation culturelle, la participation à une activité et peut être utilisé, par exemple, pour le programme « Goal-Oriented Digital Transformation ».

Une activité est une action qui est délibérément réalisée par un agent dans l’intention de produire un effet culturel, social ou matériel. Dans ce profil d’application, nous faisons référence à cet Agent en tant que Réalisateur qui peut être une Personne ou une Organisation.

Une activité peut être suivie par un participant. Cette relation s’appelle une participation. Une transaction et un billet peuvent alors y être liés. Des commentaires peuvent également être donnés sur la participation.

Pour le participant à une activité, un profil de participant générique ou personnel peut être échangé dans lequel sont contenues les questions relatives à la participation culturelle, aux caractéristiques générales et sociodémographiques.

Le Réalisateur propose l’Activité et peut effectuer la Promotion par le biais d’un Canal. Cette classe peut être spécifiée davantage dans l’Organisateur.

À propos de l’activité, il y a des informations sur quelle chose est réalisée et par quel entrepreneur, mais aussi sur les informations sur les prix et les informations sur la réservation.

Le profil d’application réutilise les termes et la façon dont ils sont appliqués à partir de la profil d’application OSLO-Organization, OSLO-Patrimoine Culturel :Evénement et OSLO-CulturalHeritage :Objet, Personne OSLO, OSLO-Consent, Bourse OSLO et OSLO-ActualAssociation.

Une description plus formelle des termes peut être trouvée dans le vocabulaire OSLO-Cultural Participation.
