This document describes an application profile, in this case OSLO Dossier - Application Profile. The application profile specifies a semantic data model covering a set of usecases. The specification consists of terms with their constraints (cardinalities, codelists, usage guidelines).

Summary

This is a basic application profile on the OSLO Dossier Vocabulary.

The conceptual information model forms the basis for the exchange of dossier data between different departments. It defines a number of abstract entities, relevant properties, and their interrelationships. This conceptual information model forms the basis for developers and database managers to further develop into a physical data model. The vocabulary provides an overview of the terminology used and the associated definitions, the basic application profile brings these terms together in a generic specification for data exchange.

To illustrate, example use cases are elaborated in this document.
