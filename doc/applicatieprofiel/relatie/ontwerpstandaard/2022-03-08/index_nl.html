Relatie

Beschrijving
Een klasse die de relatie tussen twee nodes beschrijft.
Gebruik
Een tree:Relation die aan de hand van zijn specifieke subklasse de specifieke relatie tussen twee waardes op twee tree:node weergeeft. Deze relaties worden beschreven in het pakket relatie.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: path, value.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
path URI 1 Een property path, gedefinieerd door een SHACL, die weergeeft welke zaken be√Ønvloed worden door tree:value Een SHACL propery path zoals gedefinieerd in punt 2.3.1. van de SHACL specificatie. Alle mogelijke combinaties van bijvoorbeeld shacl:alternativePath, shacl:inversePath of shacl:inLanguage uit de SHACL specificatie mogen gebruikt worden.
value URI 1 De waarde in de node waarmee de relatie gaat vergelijken om een volgende node aan te linken. Een tree:value die de contextuele waarde van de node weergeeft. Mogelijke opties hierbij zijn: een string, een integer of een link naar een andere bron waar een duidelijke regel om de waarde van noden te vergelijken geldt.
