Ce document décrit un profil d’application, dans ce cas Plages spatiales (profil d’application). Ce profil d’application répond à la question de savoir comment le modèle de domaine correspondant peut être appliqué dans la pratique. De plus, les limitations (cardinalité, listes de codes) sont expliquées et les termes correspondants (RDF) sont listés.
