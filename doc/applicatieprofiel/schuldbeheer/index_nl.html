Afhankelijkheden

Dit document beschrijft een applicatieprofiel, in dit geval OSLO Schuldbeheer - Applicatieprofiel. Dit applicatieprofiel beantwoordt de vraag over hoe het corresponderende domeinmodel in de praktijk kan toegepast worden. Daarbij worden de beperkingen (kardinaliteit, codelijsten) toegelicht en de overeenkomstige (RDF) termen opgelijst.

Wanbetalingsvergoeding

Beschrijving
Vergoeding die in rekening worden gebracht als gevolg van een wanbetaling.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: aanmaning, interest, kost, type.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
aanmaning Aanmaning 0..* Verwijzing naar een aanmaning die waarin de wanbetalingskost wordt gecommuniceerd aan de schuldenaar.
interest Percentage 0..1 De omvang van de wanbetalingsvergoeding kan bepaald worden als een percentage op bedrag van de schuldvordering dat nog niet betaald werd.
kost 0..1 De omvang van de wanbetalingsvergoeding kan worden bepaald door een vast geldbedrag.
type 1 Laat toe om de aard van de wanbetalingsvergoeding verder te specifieren Bijvoorbeeld verwijlinterest, aanmaningskost, verzendingskost

Identificator

Beschrijving
Informatie gebruikt om een object uniek te identificeren.
Gebruik
Uitgangspunt hier is dat deze string door een organisatie wordt toegekend en dat dit gebeurt volgens een welbepaald systeem.
Eigenschappen
Voor dit datatype zijn de volgende eigenschappen gedefinieerd: identificator, toegekend door, toegekend op.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
identificator GetypeerdeString 0..1 String gebruikt om het object uniek te identificeren. Type vd string slaat op het identificatiesysteem (incl de versie ervan), de string zelf op de eigenlijke identificator.
toegekend door Agent 0..1 Link naar de agent die de identificator heeft uitgegeven.
toegekend op DateTime 0..1 Tijdstip waarop de identificator werd uitgegeven.
