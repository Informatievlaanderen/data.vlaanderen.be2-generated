This document describes a vocabulary, in this case De Facto Associations - Vocabulary. A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is recorded through a label and definition. For terms defined in this domain, additional formal agreements are available. For external terms, the formal agreements are copied from the external source.

Summary

This vocabulary defines a number of terms to describe ActualAssociations.

A Fact-Finding Association is defined as an Organization that, like other Associations, serves a disinterested purpose. The difference is that a De facto Association is not registered as a legal entity, it is merely an Association because the members undertake activities together.

Actual Associations can be recognized, which gives them a formal character without being a legal entity.

In order to be able to make use of Public Services, e.g. the lending of sports equipment from a local government, it may be sufficient to provide some basic information, but sometimes this Recognition is also necessary.

The processing of the Application for Recognition and the final recognition is done by a Public Organisation. In order to obtain the Recognition, certain proofs must be provided by the Actual Association on the basis of specific criteria.

The vocabulary also makes it possible to describe who is a member of the Factual Association and/or who is responsible (and what that responsibility is).

Finally, it is possible to describe which Public Services have been requested by the Fact-Association.

The vocabulary reuses terms from the OSLO Organisation vocabulary and OSLO Service Catalogue.

4. Classes

This section provides a formal definition to each class.

Class Application for recognition

Type Class
URI https://data.vlaanderen.be/ns/FeitelijkeVerenigingen#Erkenningsaanvraag
Specialization of
  • http://purl.org/vocab/cpsv#PublicService
Definition The request for designation by a competent institution establishing that a person or institution meets certain requirements, whether temporary or not.

Class Association

Type Class
URI https://data.vlaanderen.be/ns/FeitelijkeVerenigingen#Vereniging
Specialization of
  • http://www.w3.org/ns/org#Organization
Definition An Organization that pursues a selfless goal.

Class FactualAssociation

Type Class
URI https://data.vlaanderen.be/ns/FeitelijkeVerenigingen#FeitelijkeVereniging
Specialization of
  • https://data.vlaanderen.be/ns/FeitelijkeVerenigingen#Vereniging
Definition An Association without legal personality, based on an agreement between two or more persons and/or organisations to pursue a selfless purpose within the framework of one or more well-defined activities.

Class Recognition

Type Class
URI https://data.vlaanderen.be/ns/FeitelijkeVerenigingen#Erkenning
Specialization of
  • https://data.europa.eu/m8g/Output
Definition The designation by a competent institution establishing that a person or institution meets certain requirements, whether temporary or not.
