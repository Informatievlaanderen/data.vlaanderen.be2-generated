Classes

This section gives a formal definition to each class.

Completeness

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Completeness
Definition The presence or absence of certain features, their attributes and relationships. Takes the form of commission (too much data available) or omission (missing data).
Label Completeness
Parent http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Element

ConformityResult

URI http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_ConformanceResult
Definition Information on the result of the evaluation of the value(s) obtained according to a specified conformity quality level.
Label ConformityResult
Parent http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_Result

DescriptiveResult

URI http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_DescriptiveResult
Definition A subjective evaluation of an Element in textual form.
Label DescriptiveResult
Parent http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_Result

Evaluation method

URI http://def.isotc211.org/iso19157/2013/DataQualityEvaluation#DQ_EvaluationMethod
Definition Description of the evaluation method and the procedure used.
Label Evaluation method

EvaluationMethodType

URI http://www.w3.org/2004/02/skos/core#Concept
Definition The type of method that measures data quality.
Label EvaluationMethodType

LogicalConsistency

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_LogicalConsistency
Definition The extent to which the logical rules of the data structure, attribution and relationships are met.
Label LogicalConsistency
Parent http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Element

PositionalAccuracy

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_PositionalAccuracy

Label PositionalAccuracy
Parent http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Element

Quality Element

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Element

Label Quality Element

Quality Result

URI http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_Result
Definition The result of the evaluation of the data quality.

QuantitativeResult

URI http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_QuantitativeResult
Definition The values or information about the values obtained by applying a quality measure.
Label QuantitativeResult
Parent http://def.isotc211.org/iso19157/2013/DataQualityResult#DQ_Result

SizeReference

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_MeasureReference

Label SizeReference

TemporalQuality

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_TemporalQuality
Definition The quality of the temporal attributes and temporal relationships of the data.
Label TemporalQuality
Parent http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_Element

ThematicAccuracy

URI http://def.isotc211.org/iso19157/2013/DataQuality_2#DQ_ThematicAccuracy

Label ThematicAccuracy

UsabilityElement
