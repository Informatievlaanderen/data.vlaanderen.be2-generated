Activité

Definition Acte à durée déterminée sur ou avec des entités.
Label Activité

Affaire

URI http://dbpedia.org/ontology/Case
Definition Un cas est l’ensemble du travail effectué pour arriver à une décision administrative ou commerciale. Une affaire se précipite généralement dans une collection de documents.
Label Affaire
Parent http://dbpedia.org/ontology/UnitOfWork

Agent

URI http://www.w3.org/ns/prov#Agent
Definition Empty
Label Agent

URI http://www.w3.org/ns/prov#Association

Label Agent (en rôle)

Collection

URI http://www.w3.org/ns/prov#Collection
Definition Une collection est une entité qui structure certains composants, qui sont eux-mêmes des entités. Ces composants feraient partie des collections.
Label Collection

Contexte de l’activité

URI https://data.vlaanderen.be/ns/dossier#ActiviteitContext

Label Contexte de l’activité
Parent http://www.w3.org/ns/prov#EntityInfluence

Entité

URI http://www.w3.org/ns/prov#Entity
Definition Quelque chose avec une existence claire et indépendante.
Label Entité

EntitéInfluence

URI http://www.w3.org/ns/prov#EntityInfluence
Definition La capacité d’une entité à avoir un effet sur le caractère, le développement ou le comportement d’une autre entité par l’utilisation, le début, la fin, la distraction ou autre.
Label EntitéInfluence

Lime

Parent http://www.w3.org/ns/prov#Entity

Partie

http://www.w3.org/ns/prov#Entity

Projet

URI http://dbpedia.org/ontology/Project
Definition Un projet est un effort temporaire visant à atteindre certains objectifs.
Label Projet
Parent http://dbpedia.org/ontology/UnitOfWork

Représentation

URI http://www.w3.org/ns/dcat#Distribution
Definition Une représentation est une représentation particulière d’une pièce ou d’une partie de celle-ci.
Label Représentation

Ressource

URI http://www.w3.org/2000/01/rdf-schema#Resource
Definition Entité à partir de laquelle toutes les entités sont dérivées.
Label Ressource

Série

URI https://data.vlaanderen.be/ns/dossier#Serie
Definition Une série est une collection de pièces qui partagent des traits communs.
Label Série
Usage note Pour les attributs, voir le modèle « Archives numériques Flandre »
Parent http://www.w3.org/ns/prov#Collection

Unité de travail

URI http://dbpedia.org/ontology/UnitOfWork
Definition Cette entité est censée transmettre la notion d’une quantité de travail. Elle diffère de l’activité en ce qu’elle a une fin claire et qu’elle est mesurée.
Label Unité de travail

Étape de la procédure

URI https://data.vlaanderen.be/ns/dossier#Procedurestap
Definition Une étape procédurale est l’ensemble du travail effectué dans le cadre d’une partie chronologique, logique, organisationnelle ou juridiquement déterminée d’une procédure.
Label Étape de la procédure
Parent http://dbpedia.org/ontology/UnitOfWork
