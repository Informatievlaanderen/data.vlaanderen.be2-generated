Activiteit

URI http://www.w3.org/ns/prov#Activity
Definition Handeling van bepaalde duur op of met entiteiten.
Label Activiteit

Activiteitcontext

URI https://data.vlaanderen.be/ns/dossier#ActiviteitContext
Definition Context van een relatie tussen dossierstukken. Deze context maakt de samenhang van dossierstukken duidelijk in het kader van één bepaalde activiteit.
Label Activiteitcontext
Parent http://www.w3.org/ns/prov#EntityInfluence

Agent

URI http://www.w3.org/ns/prov#Agent
Definition Empty
Label Agent

Agent (in rol)

URI http://www.w3.org/ns/prov#Association
Definition Een toewijzing van verantwoordelijkheid aan een agent voor een activiteit, wat aangeeft dat de agent een rol in de activiteit had.
Label Agent (in rol)

Collectie

URI http://www.w3.org/ns/prov#Collection
Definition Een collectie is een entiteit die structuur geeft aan sommige bestanddelen, die zelf entiteiten zijn. Deze bestanddelen zouden deel uitmaken van de collecties.
Label Collectie

Dossier

URI https://data.vlaanderen.be/ns/dossier#Dossier
Definition Samenhangend geheel van stukken die betrekking hebben op een bepaalde zaak.
Label Dossier
Parent http://www.w3.org/ns/prov#Collection

Eenheid Van Werk

URI http://dbpedia.org/ontology/UnitOfWork
Definition Deze entiteit is bedoeld om de notie van een hoeveelheid werk over te brengen. Het verschilt van Activiteit in die zin dat het een duidelijk einde heeft en gemeten wordt.
Label Eenheid Van Werk

Entiteit

URI http://www.w3.org/ns/prov#Entity
Definition Iets met een duidelijk en onafhankelijk bestaan.
Label Entiteit

EntiteitInvloed

URI http://www.w3.org/ns/prov#EntityInfluence
Definition Het vermogen van een entiteit om een effect te hebben op het karakter, de ontwikkeling of het gedrag van een andere entiteit door middel van gebruik, start, einde, afleiding of andere.
Label EntiteitInvloed

Procedurestap

URI https://data.vlaanderen.be/ns/dossier#Procedurestap
Definition Een procedurestap is het geheel aan werk gedaan binnen een specifiek chronologisch, logisch, organisatorisch of wettelijk bepaald onderdeel van een procedure.
Label Procedurestap
Parent http://dbpedia.org/ontology/UnitOfWork

Project

URI http://dbpedia.org/ontology/Project
Definition Een project is een tijdelijke inspanning om bepaalde doelstellingen te bereiken.
Label Project
Parent http://dbpedia.org/ontology/UnitOfWork

Representatie

URI http://www.w3.org/ns/dcat#Distribution
Definition Een representatie is een bepaalde weergave van een stuk(onderdeel).
Label Representatie

Resource

URI http://www.w3.org/2000/01/rdf-schema#Resource
Definition Entiteit waarvan alle entiteiten zijn afgeleid.
Label Resource

Serie

URI https://data.vlaanderen.be/ns/dossier#Serie
Definition Een serie is een collectie van stukken die gemeenschappelijke eigenschappen deelt.
Label Serie
Usage note Voor attributen, zie model "Digitaal Archief Vlaanderen"
Parent http://www.w3.org/ns/prov#Collection

Stuk

URI https://data.vlaanderen.be/ns/dossier#Stuk
Definition Eenheid van bewijs ter ondersteuning van een feit.
Label Stuk
Parent http://www.w3.org/ns/prov#Entity

Stukonderdeel

URI https://data.vlaanderen.be/ns/dossier#Stukonderdeel
Definition Een apart onderdeel van een groter stuk.
Label Stukonderdeel
Parent http://www.w3.org/ns/prov#Entity
https://data.vlaanderen.be/ns/dossier#Stuk

Zaak

URI http://dbpedia.org/ontology/Case
Definition Een zaak is het geheel aan werk gedaan om tot een bestuurlijke of zakelijke beslissing te komen. Een zaak slaat doorgaans neer in een verzameling documenten.
Label Zaak
Parent http://dbpedia.org/ontology/UnitOfWork
