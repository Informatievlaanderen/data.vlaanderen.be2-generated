Classes

This section gives a formal definition to each class.

Activity

URI http://www.w3.org/ns/prov#Activity
Definition Fixed-term act on or with entities.
Label Activity

Activity context

URI https://data.vlaanderen.be/ns/dossier#ActiviteitContext
Definition Context of a relationship between file documents. This context makes it clear that the documents in the case are consistent in the context of a particular activity.
Label Activity context
Parent http://www.w3.org/ns/prov#EntityInfluence

Agent

URI http://www.w3.org/ns/prov#Agent
Definition Empty
Label Agent

Agent (in role)

URI http://www.w3.org/ns/prov#Association
Definition An assignment of responsibility to an agent for an activity, indicating that the agent had a role in the activity.
Label Agent (in role)

Collection

URI http://www.w3.org/ns/prov#Collection
Definition A collection is an entity that gives structure to certain components, which are themselves entities. These components would be part of the collections.
Label Collection

Entity

URI http://www.w3.org/ns/prov#Entity
Definition Something with a clear and independent existence.
Label Entity

EntityInfluence

URI http://www.w3.org/ns/prov#EntityInfluence
Definition The ability of one entity to have an effect on the character, development, or behavior of another entity through use, start, end, distraction, or other.
Label EntityInfluence

File

URI https://data.vlaanderen.be/ns/dossier#Dossier
Definition Coherent set of documents relating to a particular case.
Label File
Parent http://www.w3.org/ns/prov#Collection

Matter

URI http://dbpedia.org/ontology/Case
Definition A case is the whole of the work done to arrive at an administrative or business decision. A case usually precipitates in a collection of documents.
Label Matter
Parent http://dbpedia.org/ontology/UnitOfWork

Part

URI https://data.vlaanderen.be/ns/dossier#Stukonderdeel
Definition A separate part of a larger piece.
Label Part
Parent https://data.vlaanderen.be/ns/dossier#Stuk
http://www.w3.org/ns/prov#Entity

Piece

URI https://data.vlaanderen.be/ns/dossier#Stuk
Definition Unit of evidence in support of a fact.
Label Piece
Parent http://www.w3.org/ns/prov#Entity

Procedure step

URI https://data.vlaanderen.be/ns/dossier#Procedurestap
Definition A procedural step is the entirety of work done within a specific chronological, logical, organizational, or legally determined part of a procedure.
Label Procedure step
Parent http://dbpedia.org/ontology/UnitOfWork

Project

URI http://dbpedia.org/ontology/Project
Definition A project is a temporary effort to achieve certain objectives.
Label Project
Parent http://dbpedia.org/ontology/UnitOfWork

Representation

URI http://www.w3.org/ns/dcat#Distribution
Definition A representation is a particular representation of a piece or part of it.
Label Representation

Resource

URI http://www.w3.org/2000/01/rdf-schema#Resource
Definition Entity from which all entities are derived.
Label Resource

Series

URI https://data.vlaanderen.be/ns/dossier#Serie
Definition A series is a collection of pieces that shares common traits.
Label Series
Usage note For attributes, see model "Digital Archive Flanders"
Parent http://www.w3.org/ns/prov#Collection

Unity of work

URI http://dbpedia.org/ontology/UnitOfWork
Definition This entity is meant to convey the notion of a quantity of work. It differs from Activity in that it has a clear end and is measured.
Label Unity of work
