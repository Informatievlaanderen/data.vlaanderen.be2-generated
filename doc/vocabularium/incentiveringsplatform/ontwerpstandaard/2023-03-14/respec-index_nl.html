Classes

Agent

URI http://purl.org/dc/terms/Agent
Definition Iemand die of iets dat kan handelen of een effect kan teweeg brengen.
Label Agent

Beloning

URI https://data.vlaanderen.be/ns/incentiveringsplatform#Beloning
Definition De bekrachtiging die bij een Stimulansaanbod hoort. Dit is wat zal ontvangen worden bij de consumptie van dit Stimulansaanbod. Dit is niet de effectief ontvangen beloning, maar de beschrijving van wat je zal ontvangen.
Label Beloning
Parent http://data.europa.eu/m8g/Output

Beloningsontvangst

URI https://data.vlaanderen.be/ns/incentiveringsplatform#Beloningsontvangst
Definition De effectieve beloning die ontvangen is bij de consumptie van een Stimulansaanbod.
Label Beloningsontvangst
Parent http://data.europa.eu/m8g/Output

Bewijs

URI http://data.europa.eu/m8g/Evidence
Definition De beschrijving van wat men moet voorleggen om aan een Stimulansaanbod te kunnen deelnemen en hiervoor een Beloning te ontvangen (= Beloningsontvangst).
Label Bewijs

Bewijslevering

URI http://data.europa.eu/m8g/Evidence

Label Bewijslevering
