Classes

This section gives a formal definition to each class.

ChangeableObject

URI https://data.vlaanderen.be/ns/ldes#VeranderlijkObject
Definition A changeable object is an object whose state can change after it has been created.
Label ChangeableObject
Usage note A changeable object is used as the basic element of a dataset.
Parent https://data.vlaanderen.be/ns/ldes#CollectieLid

CollectieLid

URI https://data.vlaanderen.be/ns/ldes#CollectieLid
Definition Empty
Label CollectieLid

Collection

URI https://w3id.org/tree#Collection
Definition A collection is a collection of objects. These members all conform to an imposed shape.
Label Collection

CollectionMember

URI https://data.vlaanderen.be/ns/ldes#CollectieLid
Definition An object that is part of a collection.
Label CollectionMember
Usage note Can be further specialized according to the type of collection, for example if the collection is an event stream, the collection members are unchanging objects.

Data service

URI http://www.w3.org/ns/dcat#DataService
Definition A collection of operations that either provide access to one or more datasets or have a data processing purpose.
Label Data service

DataServiceLDES

URI https://data.vlaanderen.be/ns/ldes#DataServiceLDES
Definition A collection of operations that either provide access to one or more Linked Data Event Streams.
Label DataServiceLDES
Usage note By default, the protocol here is that of the Linked Data Event Stream.
Parent http://www.w3.org/ns/dcat#DataService

Dataset

URI https://www.w3.org/ns/dcat#Dataset
Definition A set of data, published or compiled by a single agent, and available for access or download in one or more modes of presentation.
Label Dataset

Dataset

URI https://www.w3.org/ns/dcat#Dataset
Definition A collection of data, published or compiled by a single agent, and available for access or download in one or more modes of presentation.
Label Dataset

Derivative Collection

URI https://data.vlaanderen.be/ns/ldes#AfgeleideCollectie
Definition A collection that is derived from another collection.
Label Derivative Collection
Parent https://w3id.org/tree#Collection

DurationAgoPolicy

URI https://w3id.org/ldes#DurationAgoPolicy
Definition A DurationAgoPolicy uses a tree:value to indicate whether the timestamp can be redefined by the policy itself.
Label DurationAgoPolicy
Usage note A DurationAgoPolicy uses a tree:view to indicate whether the timestamp can be redefined by the policy itself.
Parent https://w3id.org/ldes#RetentionPolicy

EventStream

URI https://w3id.org/ldes#EventStream
Definition A collection of immutable objects (such as version objects & sensor observations). These objects follow imposed shape.
Label EventStream
Parent https://w3id.org/tree#Collection
https://www.w3.org/ns/dcat#Dataset

EventstreamView

URI https://data.vlaanderen.be/ns/ldes#EventstreamView
Definition The event stream view connects the collection to the root node in such a way that all members of the collection can be found.
Label EventstreamView

ImmutableObject

URI https://data.vlaanderen.be/ns/ldes#OnveranderlijkObject
Definition An Immutable Object is the basic element of a Linked Data Event Stream.
Label ImmutableObject
Usage note An immutable object is the basic element of an event stream.
Parent https://data.vlaanderen.be/ns/ldes#CollectieLid

LatestVersionSubset

URI https://w3id.org/ldes#LatestVersionSubset
Definition The latest version of an asset/subset that complies with a particular policy.
Label LatestVersionSubset
Parent https://w3id.org/ldes#RetentionPolicy

Node

URI https://w3id.org/tree#Node
Definition A node is a page that can contain relationships to other nodes.
Label Node

Object

URI http://www.w3.org/ns/prov#Entity
Definition Entity from which all entities are derived.
Label Object

Observation

URI http://def.isotc211.org/iso19156/2011/Observation#OM_Observation
Definition Determining the value of a particular feature of an object at a specific time or between two points in time.
Label Observation
Usage note NOTE: This class is described in more detail in the OSLO <a href='https://data.vlaanderen.be/standaarden/standaard-in-ontwikkeling/vocabularium-observaties-en-metingen.html'> Observations and Measurements </a>application profile.

Relation

URI https://w3id.org/tree#Relation
Definition A class that describes the relationship between two nodes.
Label Relation
Usage note There are specific subclasses that describe a specific type of relationship between the nodes. These specific relationships are described on <a href="https://treecg.github.io/specification/#Relation">the TREE specification</a>.

Retention policy

URI https://w3id.org/ldes#RetentionPolicy
Definition A retention policy is a policy according to which elements (i.e. data) will be deleted.
Label Retention policy
Usage note Some examples of retention policies are DurationAgoPolicy and LatestVersionSubset.

RootNode

URI https://data.vlaanderen.be/ns/ldes#RootNode
Definition A RootNode is a home page, which can already contain members, through whose linked pages all members can be found.
Label RootNode
Usage note The root node is referenced from the event stream, preferably via an event stream view.
Parent https://w3id.org/tree#Node

Shape

URI http://www.w3.org/ns/shacl#NodeShape
Definition A node shape is a shape in the shapes graph that is not the subject of a triple with sh:path as its predicate. It is recommended, but not required, for a node shape to be declared as a SHACL instance of sh:NodeShape. SHACL instances of sh:NodeShape cannot be
Label Shape
Usage note A shape is either described by a URi or inline. Both options are possible.

Snapshot

URI https://data.vlaanderen.be/ns/ldes#Snapshot
Definition An event stream of versions where only the latest version is retained compared to a specified date.
Label Snapshot
Parent https://w3id.org/ldes#EventStream
https://www.w3.org/ns/dcat#Dataset

Version

URI https://data.vlaanderen.be/ns/generiek#Versie
Definition Represents an entity in one specific time period.
Label Version
Usage note NOTE: This class is described more extensively in <a href='https://data.vlaanderen.be/ns/generiek#Versie'> OSLO Generic </a>.

VersionEventStream

URI https://data.vlaanderen.be/ns/ldes#VersieEventStream
Definition An event stream of which the members are versions.
Label VersionEventStream
Parent https://w3id.org/ldes#EventStream

VersionMaterialization

URI https://data.vlaanderen.be/ns/ldes#VersieMaterialisatie
Definition A snapshot where the version identifiers have been replaced by object identifiers.
Label VersionMaterialization
Usage note For this reason (see definition), the members of the collection are changeable rather than unchanging objects.
Parent https://w3id.org/tree#Collection
https://www.w3.org/ns/dcat#Dataset
