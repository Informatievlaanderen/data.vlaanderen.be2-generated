Dependencies

This document describes a vocabulary, in this case Local Economy - Vocabulary. A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is recorded through a label and definition. For terms defined in this domain, additional formal agreements are available. For external terms, the formal agreements are copied from the external source.

Summary

The OSLO-Local Economy vocabulary defines terms that are applicable to exchange data about a local economy. More specifically, it is about visitor measurements, transactions and all factors in the environment that can have an impact on the local economy. The vocabulary provides an overview of the terminology used and the associated definitions.

2. Conformity

An exchange of data, by whatever means, is subject to this vocabulary when it uses the terminology (classes and properties) in a manner consistent with the semantics established in the latest version of the specification (domain, scope, definition and usage) and does not use terminology from other vocabularies as an alternative to the proposed terminology included in this vocabulary.

4. Classes

Class Crowd Characteristic

Type Class
URI https://data.vlaanderen.be/ns/lokale-economie#Druktekenmerk
Specialization of
  • http://def.isotc211.org/iso19109/2005/GeneralFeatureModel#GF_PropertyType
