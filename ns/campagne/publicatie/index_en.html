This document describes a vocabulary, in this case Publication advertisement: Vocabulary. A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is recorded through a label and definition. For terms defined in this domain, additional formal agreements are available. For external terms, the formal agreements are copied from the external source.

Summary

The vocabulary of OSLO Publication-Advertisement is a collection of all the different elements that are important for standardizing the placement of advertisements and publications

Most of the terms were mapped to several existing standards: CIDOC-CRM and FRBRoo.

In addition, this vocabulary defines some of its own terms. These can be found under Classes and Properties, while the mapped terms are listed under External Terminology.

Status of this document

This section describes the status of this document at the time of publication. It is possible that newer versions have already been released in the meantime.


A list of the most recent vocabularies can be found at http://data.vlaanderen.be/ns


This document is an official specification of a vocabulary and provides a shared conceptual framework for certain concepts. This specification serves to clarify the definition, use, domain, and scope of the RDF terms within this domain. The terms defined in this vocabulary are persistent and will therefore never disappear, nor will the definitions change except to further clarify an existing definition. However, terms may be labeled as obsolete and replaced in newer versions of this specification.

2. Conformity

An exchange of data, by whatever means, is subject to this vocabulary when it uses the terminology (classes and properties) in a manner consistent with the semantics established in the latest version of the specification (domain, scope, definition and usage) and does not use terminology from other vocabularies as an alternative to the proposed terminology included in this vocabulary.
