AfgeleideCollectie

URI https://data.vlaanderen.be/ns/ldes#AfgeleideCollectie
Definition Een collectie die afgeleid is van een andere collectie.
Label AfgeleideCollectie
Parent https://w3id.org/tree#Collection

Collectie

URI https://w3id.org/tree#Collection
Definition Een collectie is een verzameling van objecten. Deze leden voldoen allen aan een opgelegde shape.
Label Collectie

CollectieLid

URI https://data.vlaanderen.be/ns/ldes#CollectieLid
Definition Empty
Label CollectieLid

CollectieLid

URI https://data.vlaanderen.be/ns/ldes#CollectieLid
Definition Een object dat deel uitmaakt van een collectie.
Label CollectieLid
Usage note Kan verder gespecialiseerd worden in functie van het soort collectie, bijvoorbeeld als de collectie een event stream is zijn de collectieleden onveranderlijke objecten.

DataServiceLDES

URI https://data.vlaanderen.be/ns/ldes#DataServiceLDES
Definition Een verzameling van operaties die ofwel toegang bieden tot één of meer Linked Data Event Streams.
Label DataServiceLDES

Dataservice

Label Dataservice

Dataset

URI https://www.w3.org/ns/dcat#Dataset

DurationAgoPolicy

Retentiebeleid

URI https://w3id.org/ldes#RetentionPolicy

Label Retentiebeleid
Usage note Enkele voorbeelden van retentiebeleiden zijn DurationAgoPolicy en LatestVersionSubset.

RootNode

URI https://data.vlaanderen.be/ns/ldes#RootNode

Usage note Naar de rootnode wordt verwezen vanuit de eventstream bij voorkeur via een eventstream-view.

Label Shape
Usage note Een shape wordt ofwel door een URi beschreven ofwel inline. Beide opties zijn mogelijk.
