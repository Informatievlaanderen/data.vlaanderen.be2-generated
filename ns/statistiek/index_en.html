This document describes a vocabulary, in this case OSLO Statistics (vocabulary). A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is recorded through a label and definition. For terms defined in this domain, additional formal agreements are available. For external terms, the formal agreements are copied from the external source.

Summary

The OSLO Statistics vocabulary contains terms that can be used to describe Statistical Observations, i.e. Observations that result in a Statistic.

Statistical Observation is a specialization of the Observation class from the OSLO Observations and Measurements vocabulary.

We based the Statistics class on the UNCERT model.

By Statistics we mean figures that describe the characteristics of a collection of data.

It can be about summary measures such as the mean, the median, a total or an index or about measures that say something about the distribution of the data such as the minimum or maximum, the standard deviation etc. or about the probability that a value occurs or is exceeded.

Some subclasses are provided in case additional information is needed to describe the Statistic, e.g. if it relates to a Quantile, a Moment or a Discrete or ordinary Probability.

4. Classes

This section provides a formal definition to each class.

Class DiscreteProbability

Type Class
URI https://data.vlaanderen.be/ns/statistiek#DiscreteProbabiliteit
Specialization of
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition Statistics that describe how likely a variable is to fall within a certain category.

Class Moment

Type Class
URI https://data.vlaanderen.be/ns/statistiek#Moment
Specialization of
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition Statistics that indicate how the values are distributed relative to a central measure.

Class Probability

Type Class
URI https://data.vlaanderen.be/ns/statistiek#Probabiliteit
Specialization of
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition Metric that describes how likely a variable is to exceed a certain threshold (or not).

Class Quantile

Type Class
URI https://data.vlaanderen.be/ns/statistiek#Kwantiel
Specialization of
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition Statistics that relate to a particular quantile.

Class StatisticalObservation

Type Class
URI https://data.vlaanderen.be/ns/statistiek#StatistischeObservatie
Specialization of
  • http://def.isotc211.org/iso19156/2011/Observation#OM_Observation
Definition Observation whose result is a Statistic.

Class Statistics

Type Class
URI https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition A piece of information that describes a characteristic of a dataset or variable.

Class Threshold

Type Class
URI https://data.vlaanderen.be/ns/statistiek#Drempelwaarde
Definition Value that indicates when a boundary is crossed.

5. Properties

This section provides a formal definition to each property.

Property category

Type Property
URI https://data.vlaanderen.be/ns/statistiek#categorie
Domain https://data.vlaanderen.be/ns/statistiek#DiscreteProbabiliteit
Range http://www.w3.org/2000/01/rdf-schema#Resource
Definition The category for which the probability is determined.
Usage E.g. oak for the previous e.g.

Property level

Type Property
URI https://data.vlaanderen.be/ns/statistiek#niveau
Domain https://data.vlaanderen.be/ns/statistiek#Kwantiel
Range http://www.w3.org/2001/XMLSchema#xdouble
Definition Number of the quantity to which the Statistics apply.
Usage E.g. 1 if the Statistics refer to the first quartile.

Property operator

Type Property
URI https://data.vlaanderen.be/ns/statistiek#operator
Domain https://data.vlaanderen.be/ns/statistiek#Drempelwaarde
Range http://www.w3.org/2004/02/skos/core#Concept
Definition Comparison operator.
Usage In the aforementioned example, the threshold would be exceeded with values 'greater than or equal to' 10m.

Property order

Type Property
URI https://data.vlaanderen.be/ns/statistiek#orde
Domain https://data.vlaanderen.be/ns/statistiek#Moment
Range http://www.w3.org/2001/XMLSchema#integer
Definition Order of the Moment.
Usage The mean is considered to be a moment of order 1, variation and standard deviation of order 2, order 3 for the skew of the distribution, and order 4 for the degree to which outlierts widen or narrow the distribution.

Property result

Type Property
URI https://data.vlaanderen.be/ns/statistiek#resultaat
Domain https://data.vlaanderen.be/ns/statistiek#StatistischeObservatie
Range https://data.vlaanderen.be/ns/statistiek#Statistiek
Definition The value of the observed Characteristic as a result of the Statistical Observation.

Property threshold

Type Property
URI https://data.vlaanderen.be/ns/statistiek#drempelwaarde
Domain https://data.vlaanderen.be/ns/statistiek#Probabiliteit
Range https://data.vlaanderen.be/ns/statistiek#Drempelwaarde
Definition The value that is exceeded.
Usage E.g. 10m for the previous e.g.
