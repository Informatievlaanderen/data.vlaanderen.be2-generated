Afhankelijkheden

Dit document beschrijft een vocabularium, in dit geval OSLO Statistiek (vocabularium). Een vocabularium is een verzameling van herbruikbare termen, namelijk klassen en eigenschappen, waarvan hun betekenis is vastgelegd door middel van een label en definitie. Voor termen gedefinieerd in dit domein zijn er nog bijkomende formele afspraken beschikbaar. Voor externe termen worden de formele afspraken overgenomen van de externe bron.

Samenvatting

Het vocabularium OSLO Statistiek bevat termen waarmee StatistischeObservaties kunnen worden beschreven, ttz Observaties die een Statistiek als resultaat hebben.

StatistischeObservatie is een specialisatie van de klasse Observatie uit het vocabularium OSLO Observaties en Metingen.

De klasse Statistiek baseerden we op het UNCERT model.

Onder Statistiek verstaan we cijfers die de kenmerken van een collectie gegevens beschrijven.

Het kan zowel gaan over samenvattende maten zoals het gemiddelde, de mediaan, een totaal of een index of over maten die iets zeggen over de spreiding van de gegevens zoals het minimum of maximum, de standaarddeviatie etc. of over de kans dat een waarde voorkomt of overschreden wordt.

Enkele subklassen zijn voorzien voor het geval extra info nodig is om de Statistiek te beschrijven, bvb als deze betrekking heeft op een Kwantiel, een Moment of een Discrete- of gewone Probabiliteit.

Dit document is een officiële specificatie van een vocabularium en biedt een gedeeld begrippenkader voor bepaalde concepten. Deze specificatie dient om de definitie, gebruik, domein en bereik van de RDF termen binnen dit domein te verduidelijken. De termen gedefinieerd in dit vocabularium zijn persistent en zullen bijgevolg nooit verdwijnen, noch zullen de definities veranderen behalve dan om een bestaande definitie verder te verduidelijken. Termen kunnen echter wel als verouderd worden bestempeld en vervangen worden in nieuwere versies van deze specificatie.

1. Licentie

Deze specificatie van Informatie Vlaanderen is gepubliceerd onder de "Modellicentie Gratis Hergebruik - v1.0".

2. Conformiteit

Een uitwisseling van gegevens, op welke manier deze uitwisseling ook gebeurt, is conform aan dit vocabularium wanneer het de terminologie (klassen en eigenschappen) gebruikt op een manier die consistent is met de semantiek zoals opgesteld in de nieuwste versie van de specificatie (domein, bereik, definitie en gebruik) en het geen terminologie gebruikt uit andere vocabularia als alternatief voor de voorgestelde terminologie opgenomen in dit vocabularium.

4. Klassen

Deze sectie geeft een formele definitie aan elke klasse.

Klasse DiscreteProbabiliteit

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#DiscreteProbabiliteit
Specialisatie van
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definitie Statistiek die beschrijft hoe waarschijnlijk het is dat een variable binnen een bepaalde categorie valt.

Gebruik

 Bvb de kans dat de begroeiing uit eik bestaat.

Klasse Drempelwaarde

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#Drempelwaarde
Definitie Waarde die aangeeft wanneer een grens wordt overschreden.

Gebruik

 Bvb 10m voor een maximaal waterpeil.

Klasse Kwantiel

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#Kwantiel
Specialisatie van
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definitie Statistiek die betrekking heeft op een bepaald kwantiel.

Gebruik

 Onder kwantiel verstaan we de subset van data die ontstaat door de volledige set te ordenen en in gelijke aantallen op te delen. Bvb een kwartiel van een set waarnemingen bevat 25% van alle waarnemingen. De Statistiek is dan bvb het gemiddelde van alle waarden in het eerste kwartiel.

Klasse Moment

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#Moment
Specialisatie van
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definitie Statistiek die aangeeft hoe de waarden gespreid zijn tov een centrale maat.

Gebruik

 Bvb de spreiding tov het gemiddelde, uitgedrukt als de standaarddeviatie. De Statistiek is bvb de standaardddeviatie tov het gemiddelde.

Klasse Probabiliteit

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#Probabiliteit
Specialisatie van
  • https://data.vlaanderen.be/ns/statistiek#Statistiek
Definitie Statistiek die beschrijft hoe waarschijnlijk het is dat een variabele een bepaalde drempelwaarde (al dan niet) overschrijdt.

Gebruik

 Bvb de kans dat het waterpeil 10m overschrijdt.

Klasse Statistiek

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#Statistiek
Definitie Een stuk informatie dat een kenmerk van een dataset of variable beschrijft.

Klasse StatistischeObservatie

Type Klasse
URI https://data.vlaanderen.be/ns/statistiek#StatistischeObservatie
Specialisatie van
  • http://def.isotc211.org/iso19156/2011/Observation#OM_Observation
Definitie Observatie waarvan het resultaat een Statistiek is.

Gebruik
