This document describes a vocabulary, in this case OSLO Water Quality (Vocabulary). A vocabulary is a collection of reusable terms, namely classes and properties, whose meaning is recorded through a label and definition. For terms defined in this domain, additional formal agreements are available. For external terms, the formal agreements are copied from the external source.

Summary

The OSLO Water Quality vocabulary is an extension of the European ODALA Water vocabulary.

The European vocabulary provides terminology for describing water quality, to which this vocabulary adds terms.

The most important additions are additional types of observations, namely ChemicalAgensCargoObservation and HydromorphologicalIndexObservation.

In addition to the ODALA domain object Water Object, Emission and Emission Source were also provided as potential objects to be observed or sampled.

Sampling objects can now be grouped as a Sampling Object Set of type Monitoring Network that can consist of Spatial Sampling Objects of type Measurement Point.

Also important is a class Agens in order to be able to indicate the concentration or load of a chemical substance independently of the observed characteristic. For a similar reason, the Species class was also added for BioIndicator Observations.

Finally, it is also possible to characterize an Observation as a Statistical Observation.

2. Conformity

An exchange of data, by whatever means, is subject to this vocabulary when it uses the terminology (classes and properties) in a manner consistent with the semantics established in the latest version of the specification (domain, scope, definition and usage) and does not use terminology from other vocabularies as an alternative to the proposed terminology included in this vocabulary.

4. Classes

This section provides a formal definition to each class.

Class Agent

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Agens
Definition A chemical element or chemical compound (in pure form or in a mixture) as it occurs in its natural state or is the result of, used or released in a professional activity, whether intentionally or unintentionally produced.

Class ChemicalAgensCargoObservation

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#ChemischAgensVrachtObservatie
Specialization of
  • http://def.isotc211.org/iso19156/2011/Observation#OM_Observation
  • http://def.isotc211.org/iso19156/2011/Measurement#OM_Measurement
Definition Observation to assess the chemical quality of water by the amount of a particular substance carried by the water.

Class Emission

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Emissie
Specialization of
  • http://def.isotc211.org/iso19156/2011/GeneralFeatureInstance#GFI_Feature
Definition Emission of something.

Class HydromorphologicalIndexObservation

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#HydromorfologischeIndexObservatie
Specialization of
  • http://def.isotc211.org/iso19156/2011/Observation#OM_Observation
  • http://def.isotc211.org/iso19156/2011/Measurement#OM_Measurement
  • https://data.vlaanderen.be/ns/statistiek#StatistischeObservatie
Definition Observation that assesses, according to a certain scale, the ecological status of water bodies based on their hydrological and morphological characteristics.

Class Kind

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Soort
Definition Group of organisms typified by the fact that they can have fertile offspring together.

Class Measuring point

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Meetpunt
Specialization of
  • http://def.isotc211.org/iso19156/2011/SpatialSamplingFeature#SF_SpatialSamplingFeature
Definition Place where Observations take place.

Class Monitoring network

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Meetnet
Specialization of
  • http://def.isotc211.org/iso19156/2011/SamplingFeature#SF_SamplingFeatureCollection
Definition Coherent set of places where Observations are carried out.

Class ObservedWeather

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#GeobserveerdWeer
Definition An observation of weather conditions at a certain place and time.

Class Source of emissions

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#Emissiebron
Specialization of
  • http://def.isotc211.org/iso19156/2011/GeneralFeatureInstance#GFI_Feature
Definition Origin of an emission.

Class StatisticalObservation

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#StatistischeObservatie
Definition An observation according to a statistical methodology.

Class Water QualityObservationCollection

Type Class
URI https://data.vlaanderen.be/ns/waterkwaliteit#WaterkwaliteitObservatieVerzameling
Specialization of
  • https://www.w3.org/TR/vocab-ssn-ext/#sosa:ObservationCollection
Definition A collection of one or more water quality observations. The included observations share a value for one or more properties.
